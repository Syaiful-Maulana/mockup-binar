# MockUp

### clone ke dalam local kalian

```
git clone https://github.com/
```

### pindah ke file yang berhasil di clone

```
cd no3restapi
```

### lakukan npm install

```
npm install
```

### Selanjutnya copy file .env.example dan isi seluruh data dengan benar

```
cp .env.example .env
```

### silahkan run folder

```
npm run dev

```
